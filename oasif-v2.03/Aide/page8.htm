Paramétrage : Activités pédagogiques  

L’onglet « Activités » permet d’ajouter :

  • des types d’activités supplémentaires : association d’un nom d’activité, d’une couleur de la palette Web et d’un descriptif (champ texte).

Le type correspond à l'activité d'apprentissage proposée à l'apprenant. Un certain nombre de types d'activités pédagogiques sont prédéfinis dans l'outil avec pour chaque type, une couleur correspondante. Ainsi, dans la représentation visuelle « planning » les activités pédagogiques apparaissent sous la forme de rectangles colorés

  • des types de travail individuel / collectif supplémentaires pour leur travaux : par exemple « individuel, collectif , en binôme, groupe 1 & 2 », etc. associés à un descriptif (champ texte).

L'activité pédagogique peut être réalisée individuellement ou de façon collective. Un certain nombre de types de travaux sont prédéfinis dans une liste de choix (individuel, binôme, trinôme, groupe complet). Il est possible cependant d'en définir d'autres : demi-groupe, etc.
