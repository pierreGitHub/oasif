Menu Fichier  

Il permet de créer une Nouvelle formation, d’Ouvrir une formation existante, de Fermer la formation en cours, Enregistrer ou Enregistrer sous le fichier courant (celui qui est ouvert).

Ce menu permet aussi d’Exporter le module sous forme :

  • d'un fichier descriptif de valorisation du module ou de la formation (fichier HTML avec la représentation visuelle « planning » ainsi qu’un ensemble des informations sur les activités, les séquences et les modules) lisible sur les navigateurs courants
  • d'un fichier XML qui peut être lu par les plates-formes logicielles de FOAD.

Il permet également d’Imprimer :

  • La fiche descriptive du planning courant (formation ou module) sous forme d'un fichier HTML dans un navigateur,
  • la Représentation visuelle du planning courant (formation ou module),
  • la Représentation visuelle des plannings des journées planifiées que vous avez définies.

Enfin Quitter permet de mettre fin à la session OASIF en cours.

Note  

Notez qu’il est possible d’ouvrir et travailler sur plusieurs fichiers (plusieurs formations) dans une même session OASIF. Dans ce cas, il est possible de passer d’un fichier à l’autre en utilisant la zone Arborescence.

Lorsque plusieurs formations sont ouvertes simultanément, vous pouvez copier-coller des activités pédagogiques, des séquences ou des modules d'une formation à l'autre. Dans le cas d’un copier-coller d’une activité ou d’une séquence entre deux modules, voire deux formations, OASIF s’assure de l’existence des éléments dans les listes des paramètres du module et les ajoute le cas échéant.
