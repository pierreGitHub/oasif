Les propriétés de l'activité : onglet Accompagnement  

 

Tableau de synthèse des accompagnateurs de l ’activité pédagogique.

Pour modifier les caractéristiques ou supprimer les accompagnateurs définis dans le tableau de synthèse, vous devez cliquer sur la ligne correspondante dans le tableau (ce qui fait apparaître les informations correspondantes dans la partie gauche de cet écran), faire vos modifications ou non, puis cliquer sur le bouton modifier ou supprimer.
   

Zone de définition, modification et de suppression des accompagnateurs de l ’activité pédagogique.

Une fois que vous aurez saisi un accompagnateur, vous devrez cliquer sur le bouton ajouter pour le valider et le faire entrer dans le tableau de synthèse.

N'oubliez pas de cliquer sur le bouton « Ajouter » !

Zone de définition des outils utilisés pour l’accompagnement (outil visé et outil de substitution en cas de problème)
En cliquant sur + vous pouvez en définir d’autres

Pour qu’une activité « fasse réagir » la frise temporelle sur les activités distantes synchrones, l’un des outils doit avoir la caractéristique “synchrone” dans le paramétrage…

Nom de l ’accompagnateur global : on définit le contenu de ce menu grâce au Paramétrage menu Outil.
Il est possible de définir des accompagnateurs au niveau du module et de la formation.
En cliquant sur + vous pouvez en définir d’autres

Type d’accompagnement proposé pour l'activité pédagogique : tutorat pédagogique ou intervention d'un expert par ex.
En cliquant sur + vous pouvez en définir d’autres

Attitude : permet de définir globalement ce que l ’on attend de l ’accompagnateur (son attitude globale, ses tâches).
En cliquant sur + vous pouvez en définir d’autres

Descriptif de l'accompagnement : Description précise de ce que l’on attend de l ’accompagnateur (cette information n’est pas renseignée pour les apprenants).

On précise si l'accompagnement se fera en présence ou à distance de l’apprenant.

Temps de travail estimé que devra effectuer l’accompagnateur défini pour un groupe d’apprenants donné.
On peut par exemple définir que l’expert travaillera 1h pour le groupe de 12 apprenants (groupe complet), ou que le tuteur pédago. fournira 15mn de travail par binôme d’apprenants ou pour un sous groupe défini dans le paramétrage ou pour le groupe complet (par défaut).

Vous pouvez préciser ici :

1) Le temps d’accompagnement d’une activité pour un accompagnateur = (temps * le nombre total d’apprenant) / le groupe de N apprenants
Ex : temps = (30’ * 10 apprenants)/5 = 60’ d’accompagnement pour l’activité
Si le nombre maximum d’apprenants du module n’est pas renseigné, le calcul n’est pas possible ! Vous pouvez renseigner ce nombre dans les « Propriétés de la formation, onglet Descriptif ».

2) Le temps d’accompagnement d’un accompagnateur pour un groupe précis = temps d’accompagnement saisie.

En cliquant sur + vous pouvez en définir d’autres groupes
