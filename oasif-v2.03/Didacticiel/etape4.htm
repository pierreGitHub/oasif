> Etape n° 4 : Création d’un module de formation

 

Pour créer maintenant votre module de FOAD, sélectionnez la fonction « Créer un module  » dans le menu Edition.
Un rectangle correspondant au module apparaît dans le planning et la zone Propriétés se met à jour pour afficher les « Propriétés du Module  » (spécifiques au module mais non encore définies à ce stade).

Cliquez sur le dossier Module dans l’arborescence (partie gauche de l’écran) pour travailler au niveau du Module (accès possible au module à tout moment à l’aide de cette arborescence).

 
Pour obtenir plus d’info concernant le Module,voir le « Manuel d'utilisation de OASIF ».

 

 

Note  

Pour positionner directement votre module sur le planning, faites un clic droit de la souris à cet endroit en choisissant « créer un module  ».

La durée d’une formation est déterminée chronologiquement par le 1er jour de son 1er module et le dernier jour de son dernier module.

Notez l’apparition du niveau module dans l’arborescence.
